## 重 build 步骤（必须，一次性）

v0.4 引入了新 native modules，**必须执行一次完整重 build**：

```bash

LANG=en_US.UTF-8 LC_ALL=en_US.UTF-8 pod install

cd ..
LANG=en_US.UTF-8 LC_ALL=en_US.UTF-8 npx expo run:ios          # simulator
LANG=en_US.UTF-8 LC_ALL=en_US.UTF-8 npx expo run:ios --device # 真机（先在 Xcode 里 select Team）
```
